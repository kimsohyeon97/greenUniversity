### 👉🏻 CI/CD 및 배포 환경
### GitHub Actions를 활용한 자동화 빌드 및 테스트
* 코드 푸시 시 자동으로 빌드, 테스트 진행
* 빌드 성공 시 자동 배포 트리거<br><br>

### AWS EC2 인스턴스에 애플리케이션 배포
* Spring Boot 애플리케이션을 EC2에 배포하여 실제 서비스처럼 실시간 제공
* 보안 그룹 설정 및 포트 관리로 안정적인 서비스 운영<br><br>
